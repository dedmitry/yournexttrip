TrackThisNow — Favicon Pack
============================

Files:
  favicon.svg                   Scalable vector (use in all modern browsers)
  favicon-16x16.png             Browser tab
  favicon-32x32.png             Standard favicon
  favicon-48x48.png             Windows taskbar
  favicon-64x64.png             High-DPI tab
  favicon-96x96.png             Android shortcut
  favicon-128x128.png           Chrome Web Store
  apple-touch-icon.png          Apple touch icon (180x180)
  android-chrome-192x192.png    Android home screen
  favicon-256x256.png           Windows tile
  android-chrome-512x512.png    PWA splash / maskable icon
  site.webmanifest              Web app manifest (PWA)
  html-snippet.html             Paste into your <head>

Setup:
  1. Copy all files to your /public folder
  2. Paste contents of html-snippet.html into your HTML <head>
  3. Done!
